(static) flatten(object) → {Object}

Source:
  • utils.js, line 75

Flattens nested objects. Generates path string with _.

Example
flatten({
  a: {
    b: 2,
    d: {
      e: 4
    }
  },
  c: 3,
  d: [4, 5]
})
//
// returns:
// {
//   a_b: 2,
//   a_b_d_e: 4
//   c: 3,
//   d_0: 4,
//   d_1: 5
// }

(static) isTag(args) → {Boolean}

Source:
  • utils.js, line 35

Returns if a function call is a template literal call or a standart function call.

Example
import { isTag } from './utils'

function hello(args) {
  return isTag(args)
}

hello('world') // false
hello`world` // true
